# Machine-Learning-Capstone-project-_-naive-bayes
Naïve Bayes vs Logistic Regression using the Iris dataset

## Overview
This project compares two machine learning algorithms: **Naïve Bayes** and **Logistic Regression** using the **Iris dataset**.  
The aim is to see which model performs better in classifying flower species.

## Files
- `ML capstone code.py` : Python script with documented code.
- `Figure_1.png` : Plot comparing accuracy of both models.
- `Figure_2.png` `Figure_3.png`: Heatmap showing prediction results.

## How It Works
1. Load the Iris dataset.
2. Split the data: 70% training, 30% testing.
3. Train Naïve Bayes and Logistic Regression models.
4. Make predictions on test data.
5. Evaluate performance using accuracy and confusion matrices.
6. Visualize results.

## Results
- Naïve Bayes Accuracy: 0.98  
- Logistic Regression Accuracy: 1.0  
- Confusion matrices show that most predictions are correct.
